## Distinguishing the classes

- **evidence-backed** (`content_response_click`, `exposure_response_hazard`, `attrition_dropout_hazard`):
  local held-out artifact, Tier 2.
- **verified-published, transported** (`bass_diffusion`, `ultimatum_offer_response`, `trust_game_transfer`,
  `social_pressure_turnout`, `matching_donation_response`, `reputation_updating`): core-verified DOI + broad
  transport prior + `transport_widened`, Tier 4, `domain_restricted`.
- **research-encoded** (`weak_tie_transmission`, `network_targeting_seeding`, `altruistic_punishment`,
  `persuasion_minimal_effects`): verified research + formal model, no numeric transition yet, Tier 4.
- **generic fallback**: when no family answers the process, `select_tier_for_process` returns the tier-6/7
  `generic_outcome_prior` (broad prior; the terminal safety net).
